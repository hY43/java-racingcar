# 자동차 경주 게임
## 기능 요구 사항
* 사용자는 몇 대의 자동차로 게임할지 입력한다.
* 사용자는 몇 번의 이동할지 입력한다.
* 게임은 입력된 자동차 수 대로 입력된 수만큼 이동 혹은 멈춘다.
* 전진의 조건은 0~9 사이의 숫자를 랜덤으로 받아 4 이상일 경우에 이동한다.
    - 랜덤 값을 산출한다.
    - 산출된 랜덤 값으로 전진 여부를 확정한다.
* 자동차 상태를 화면에 출력한다.        

## 예시
* 3대의 자동차가 5번 움직였을 경우
```
자동차 대수는 몇 대 인가요?
3
시도할 회수는 몇 회 인가요?
5

```
